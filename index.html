3. Colors

If scores are already in a -1 to 1 range with negative values, they are used as-is. Other numeric ranges are automatically normalized to -1 to 1.

Scores between the negative and positive threshold stay grey. Lower values color words more aggressively; higher values keep more words neutral.

Text-estimated sentiment uses a lightweight embedded lexicon. It does not use an API and is not identical to Python VADER scoring.
